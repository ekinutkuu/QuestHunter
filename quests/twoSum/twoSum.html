1. Two Sum - Sum of Two Numbers

Easy - 10 points

Given two integers a and b, return the sum of these two integers.

Example 1:
Input: a = 2, b = 3
Output: 5
Explanation: The sum of 2 and 3 is 5.

Example 2:
Input: a = -1, b = 1
Output: 0
Explanation: The sum of -1 and 1 is 0.

Example 4:
Input: a = 0, b = 0
Output: 0
Explanation: The sum of 0 and 0 is 0.

Note:
You can assume that the inputs are always valid integers.
The function should return the result as an integer.
