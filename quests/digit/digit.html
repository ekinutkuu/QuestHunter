3. Finding Digit - How Many Digits a Number Has

Hard - 100 points

You can only use PHP for now! New languages will be added in the future.

Given a non-negative integer n, return the number of digits in n.

The number of digits in a non-negative integer n is the count of individual digits present in the number. For example, the number 1234 has 4 digits, and the number 7 has 1 digit.

Example 1:
Input: n = 12345
Output: 5
Explanation: The number 12345 has 5 digits.

Example 2:
Input: n = 0
Output: 1
Explanation: The number 0 has 1 digit.

Example 3:
Input: n = 987
Output: 3
Explanation: The number 987 has 3 digits.

Note: The input n is a non-negative integer.
